TODOs API Documentation

GET /api/todos

Retrieve all TODO items.

Response:
[
    {
        "id": 1,
        "title": "Example TODO",
        "completed": false
    },
    ...
]

GET /api/todos/{id}

Retrieve a specific TODO item by ID.

Response:
{
    "id": 1,
    "title": "Example TODO",
    "completed": false
}

GET /api/todos/search

Search TODO items by keyword in title.

Request Parameters:
?keyword=example
Response:
[
    {
        "id": 1,
        "title": "Example TODO",
        "completed": false
    },
    ...
]

POST /api/todos

Create a new TODO item.

Request Body:
{
    "title": "New TODO",
    "completed": false
}
Response:
{
    "id": 1,
    "title": "New TODO",
    "completed": false
}

PUT /api/todos/{id}

Update an existing TODO item by ID.

Request Body:
{
    "title": "Updated TODO",
    "completed": true
}
Response:
{
    "id": 1,
    "title": "Updated TODO",
    "completed": true
}

DELETE /api/todos/{id}

Delete a specific TODO item by ID.

Response:
{
    "status": "success"
}
